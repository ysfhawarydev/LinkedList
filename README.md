# 🔗 LinkedList Implementation in Java (OOP)

A **fully custom implementation of LinkedList in Java** built from scratch using **Object-Oriented Programming (OOP)** principles.

This project simulates how LinkedList works internally in Java, without using built-in collections, and includes all fundamental and advanced operations.

---

## ✨ Features

- ➕ Insert nodes (at beginning, end, and specific position)
- ❌ Delete nodes (by value or index)
- 🔍 Search for elements
- 📊 Display all elements
- 🔁 Traverse the list
- 📏 Get size of the list
- 🔄 Reverse the LinkedList
- ⚡ Handle edge cases (empty list, single node, invalid index)

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)
- Dynamic Memory Allocation
- Node-based Data Structure
- Pointers (References in Java)
- Algorithm Logic & Edge Case Handling
